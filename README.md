# Procédure d'installation du programme :
## Installation
## Prérequis

Python 3.10 ou supérieur

pip installé

## 1. Télécharger le projet

Aller sur le dépôt GitHub

Cliquer sur Code

Sélectionner Download ZIP

Extraire l’archive dans un dossier de votre choix

## 2. Ouvrir un terminal dans le dossier du projet
cd chemin/vers/le/projet

## 3. Installer les dépendances
pip install pillow customtkinter sqlalchemy


Dépendances utilisées :

Pillow : gestion des images

CustomTkinter : interface graphique

SQLAlchemy : ORM pour la base de données


## 4. Lancer le programme
python Accueil.py
